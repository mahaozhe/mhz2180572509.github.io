Markov Chain Monte Carlo (MCMC) Estimation of Hidden Markov Model (HMM)

Posted on 2022-03-31 Edited on 2022-11-16 In Probabilistic Machine Learning

Hidden Markov Model (HMM) contains a sequence hidden variables \(\{X_1, X_2, ..., X_T\}\) and a sequence of observable variables \(\{Y_1, Y_2, ..., Y_T\}\), shown in the following figure. The joint probability of an HMM is:

\[\begin{equation} P(X_1, X_2, ..., X_T, Y_1, Y_2, ..., Y_T) = P(X_1)\prod_{t=2}^{T}{P(X_t|X_{t-1})}\prod_{t=1}^{T}{P(Y_t|X_t)} \end{equation}\].
